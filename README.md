# GridWise – AI Smart EV Charging Scheduler

A professional, modern, and responsive web application for optimizing EV charging schedules in smart communities.

## Features
- **AI-Driven Scheduling**: Optimizes charging slots to reduce peak load.
- **Resident Panel**: Premium interface for submitting charging requests.
- **Admin Dashboard**: Real-time load monitoring and schedule management.
- **Smart Analytics**: Visualization of energy distribution and cost savings.

## Tech Stack
- **Frontend**: React (Vite)
- **Styling**: Vanilla CSS (CSS Variables)
- **Icons**: Lucide React
- **Charts**: Recharts

## Getting Started

1.  **Install Dependencies**
    ```bash
    npm install
    ```

2.  **Run Development Server**
    ```bash
    npm run dev
    ```

3.  **Open in Browser**
    Navigate to `http://localhost:5173`
